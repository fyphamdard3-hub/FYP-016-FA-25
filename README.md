## Algo Trading Bot By AI
## Overview

This project is an Algo Trading Bot for Gold (XAU/USD) trading on MetaTrader 5 (MT5). The bot uses machine learning models and real time market data to generate trading signals and execute trades automatically.

## Features

* Automated XAU/USD trading
* MetaTrader 5 integration
* Machine learning based predictions
* Real time trade execution
* Backtesting and performance analysis
* Risk management system

## Technologies

* C / C#
* Exness(Broker)
* MQL5
* MetaTrader5
* VPS


## Objective

To develop a fully autonomous trading bot that improves trading accuracy, reduces emotional bias, and performs profitable trades with minimal human intervention.
